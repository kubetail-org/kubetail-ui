## Best Practices

- Use `text-text` for primary content and headings
- Use `text-text-muted` for secondary information, descriptions, and captions  
- Use `text-text-disabled` only for truly disabled states
- Use `text-text-on-color` when text appears on colored backgrounds
- Combine font sizes with appropriate font weights for proper hierarchy
- Maintain consistent spacing between typography elements using Tailwind's space utilities
